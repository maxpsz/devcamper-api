# DevCamper API

> Backend API for bootcamps management.

## Usage

Rename "config/config.env.dist" to "config/config.env" and update the values to your own.

## Install dependencies
```
npm install
```

## Run app
```
# Run in DEV mode
npm run dev

# Run in PROD mode
npm start

# Run in PROD mode (windows)
npm run start-win
```
